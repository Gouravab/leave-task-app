# Leave Task App

A Spring Boot application for managing employees, leaves, and tasks.

## Backend

- Built with Spring Boot 3.5.13
- Java 21
- Maven

### Running the Backend

```bash
cd backend
mvn spring-boot:run
```

## Frontend

- Simple HTML/CSS/JS

### Running the Frontend

Open `frontend/index.html` in a browser.

## Usage

- Login via the frontend
- Manage employees, leaves, tasks via API

## Troubleshooting

- Ensure Java 21 is installed
- Ensure Maven is installed
